# 3. Alcance y Contexto del Sistema

El sistema ERP se considera como una caja negra que interactúa con distintos usuarios y sistemas externos. Este diagrama permite comprender el alcance del sistema y sus principales interacciones.

El Administrador de Compras utiliza el sistema para gestionar productos, proveedores y órdenes de compra. Además, el sistema ERP se comunica con sistemas externos como el sistema contable y el sistema bancario para el intercambio de información financiera.

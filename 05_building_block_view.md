# Vista de Contenedores

El sistema ERP está compuesto por los siguientes contenedores principales:

- Aplicación Web: proporciona la interfaz de usuario que permite a los administradores interactuar con el sistema.
- API Monolítica: contiene la lógica de negocio del sistema y procesa las solicitudes realizadas desde la aplicación web.
- Base de Datos: almacena toda la información relacionada con productos, proveedores y órdenes de compra.

La comunicación entre los contenedores se realiza mediante protocolos estándar como HTTPS y JDBC.

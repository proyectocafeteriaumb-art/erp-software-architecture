# Taller de Arquitectura de Software – Sistema ERP

Este repositorio contiene el desarrollo del taller de Arquitectura de Software, cuyo objetivo es analizar, diseñar y documentar un sistema ERP enfocado en el Módulo de Compras.

El trabajo incluye la definición de requisitos, historias de usuario, diagramas arquitectónicos y documentación técnica utilizando el modelo C4 y UML.

## Contenido del Proyecto

- Documentación del sistema en la carpeta `docs`
- Diagramas generados con PlantUML en la carpeta `docs/images`
- Historias de usuario y criterios de aceptación para el Módulo de Compras
